# CGRU-OCR
OCR using Convolution and GRU(Recurrent Neural Network).This repository used pytorch framework to build the OCR.

## Requirements
pytorch>1.6.0</br>
torchvision>0.7.0
```bash
pip install -r requirments.txt
```
